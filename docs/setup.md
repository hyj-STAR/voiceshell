# 安装与会议声音设置 / Setup

## 1. 安装 VoiceShell
从 [voiceshell.tech/downloads](https://voiceshell.tech/downloads/) 下载对应平台版本并安装。
- macOS 为已签名 + 公证版本,正常双击安装即可。

## 2. 安装虚拟声卡（用于会议直连）
VoiceShell 通过虚拟声卡把"你的声音"送进会议软件。请先安装 **VB-Cable**：
- 它会提供 **CABLE Input**（输入）和 **CABLE Output**（输出）两个虚拟设备。

## 3. 路由设置
1. 在 VoiceShell 里,把输出设备选为 **CABLE Input**。
2. 在会议软件（腾讯会议 / 飞书 / Zoom 等）里,把**麦克风**选为 **CABLE Output**。
3. 这样你近耳语说话 → VoiceShell 用你的声音正常音量复述 → 进入会议。
